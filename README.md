# BitWave Web

Sitio oficial de BitWave — empresa de intercambio de criptoactivos en Colombia.

## Stack
Next.js 15 · TypeScript · Tailwind CSS v4 · MDX · next-intl · Vercel

## Desarrollo local

```bash
npm install
npm run dev
```

## Publicar un post de blog

1. Crear `content/blog/mi-post.mdx` con frontmatter
2. Commit + push → Vercel despliega automáticamente

## Variables de entorno

Ver `.env.example`. No se requieren variables externas.

## Deploy

Vercel via GitHub. Push a `main` → deploy a producción.
Dominio: bitwaveco.com
